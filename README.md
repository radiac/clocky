# clocky

A little desktop alarm clock

![Two alarms are set, they are red because they are imminent](https://github.com/radiac/clocky/raw/master/screenshot.png)


## Development

Testing web:
```
cd repo/src
python -mhttp.server 5000
```

Testing app:
```
cd repo/
cargo tauri dev
```

Building:
```
cd repo/
cargo tauri build
```
